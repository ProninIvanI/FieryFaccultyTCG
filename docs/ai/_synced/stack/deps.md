# Зависимости стека web-ccg

Версия: 1.0  
Дата: 2026-03-15

## 1. Frontend

- **React 18**:
  - основной UI‑фреймворк;
  - функциональные компоненты и хуки.
- **TypeScript**:
  - строгая типизация всего клиентского кода;
  - `strict: true` и связанные опции.
- **Vite**:
  - быстрый dev‑server и сборка фронтенда;
  - конфигурация минималистична, без избыточной магии.
- **react-router-dom**:
  - маршрутизация между основными страницами:
    - запуск матча;
    - декбилдер;
    - просмотр карт;
    - новости и дебаг.
- **CSS Modules**:
  - инкапсуляция стилей на уровне компонентов;
  - предотвращение конфликтов имён классов.

---

## 2. Game-core

- **TypeScript**:
  - реализация ядра игры;
  - единый язык с фронтендом и сервером.
- **Детерминированный RNG**:
  - генератор случайных чисел с seed;
  - все случайные решения, влияющие на исход матча, проходят через него.
- **Паттерны**:
  - Event System для уведомления о событиях в матче;
  - Command Pattern для описания действий игроков и системы;
  - State Machine для управления фазами игры.

---

## 3. Backend

- **Node.js + TypeScript**:
  - серверная платформа и язык;
  - строгая типизация и общее поле типов с `game-core`.
- **ws (WebSocket)**:
  - реализация WebSocket‑сервера;
  - управление подключениями и обменом сообщениями с клиентами.

---

## 4. Инструменты качества и инфраструктуры

- **ESLint**:
  - линтер для TypeScript‑кода (frontend, backend, game-core);
  - включает правила для React и Node.
- **TypeScript strict**:
  - обязательный `typecheck` в CI;
  - ошибки типов блокируют сборку.
- **Docker / docker-compose (опционально)**:
  - контейнеризация сервера и зависимостей;
  - `docker-compose` применяется для локального запуска и дебага, а не как обязательная часть деплоя.
