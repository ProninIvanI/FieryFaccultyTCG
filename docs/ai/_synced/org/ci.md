# CI/CD

Версия: 1.0
Последнее обновление: 2026-02-20

Контракт CI/CD пайплайна. Обязательные стадии и артефакты для всех проектов.

---

## Обязательные стадии

| Стадия | Назначение |
|--------|------------|
| `analysis` | Линтеры + сканеры безопасности |
| `test` | Юнит-тесты, генерация coverage |
| `build` | Сборка приложения |

---

## Обязательные артефакты

- **junit** - результаты тестов в формате JUnit XML
- **coverage** - отчёт покрытия (HTML и/или текстовый)

---

## Дополнительные стадии (при необходимости)

| Стадия | Назначение |
|--------|------------|
| `docker` | Сборка и публикация Docker-образа |
| `check` | Проверка работоспособности (liveness/readiness) |

---

## Требования

- CI Runner с доступом к Docker (если используется Docker-стадия).
- Переменные окружения для целевых окружений настраиваются в CI/CD settings.
- Конкретные версии образов, порты и job-имена определяются в проектном `.gitlab-ci.yml`.

---

## Инструменты проверки

- Линтеры и сканеры безопасности входят в обязательный набор проверок.
- Все инструменты запускаются единой командой из `justfile`.
